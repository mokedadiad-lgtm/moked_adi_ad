# גופנים ל־PDF (אופציונלי)

כדי שה־PDF יציג עברית עם גופן Heebo, התקן:

```bash
npm install @fontsource/heebo
```

המערכת תטען את הגופן מ־`node_modules/@fontsource/heebo/files/`.

**אלטרנטיבה:** להעתיק לכאן את הקבצים:
- `heebo-hebrew-400-normal.woff2`
- `heebo-hebrew-700-normal.woff2`

(ניתן להוריד מ־[Google Fonts Heebo](https://fonts.google.com/specimen/Heebo) או מ־[fontsource](https://fontsource.org/fonts/heebo).)
