# WHATSAPP BOT (INBOUND) - OFFICIAL TEXTS & BUTTONS

מטרת המסמך: סט תבניות רשמי לבוט הוואטסאפ (FSM) כולל טקסטים, וכפתורים בכל שלב.
המסמך מיועד לעריכה ידנית ע"י הצוות; בעתיד ניתן לחבר את הקוד לקריאת טקסטים מכאן.

## משתנים (להחלפה בתוך הטקסטים)
- `{{asker_gender}}` -> `זכר` / `נקבה`
- `{{asker_age}}` -> גיל כמספר
- `{{title}}` -> כותרת
- `{{content}}` -> תוכן השאלה (טקסט מלא, כולל שורות)
- `{{response_type}}` -> `short` / `detailed` (או קצר/מורחב לפי מה שתבחרו להציג)
- `{{publication_consent}}` -> `publish` / `blur` / `none`
- `{{delivery_preference}}` -> `whatsapp` / `email` / `both`
- `{{asker_email}}` -> אימייל אם נאסף

## UX כללי (אופציונלי)
- קלט שלא מתאים לכפתורים: להחזיר הודעת תיקון קצרה + להציג כפתורים של אותו שלב.
- תוכן שאלה בכמה הודעות: לצבור ב-`context.body_parts[]` ואז בסיום לשמור כטקסט רציף (למשל join עם שורה חדשה).
- "עד 4 שינויים": הכוונה למספר פעולות עריכה (גם אם אותו שדה), כפי שסוכם.

## הערה טכנית (למפתחי ה-FSM)
- מומלץ לייצר מזהה הודעה/קלט (`provider_message_id`) ולמנוע עיבוד כפול.
- בזמן `waiting_admin_approval` לפי הסיכום הנוכחי:
  - אם זו לא "שאלה חדשה" אלא בקשת שינוי במסגרת ההמתנה: לא מעדכנים אוטומטית, רק מציגים לאדמין דרך Inbox.
